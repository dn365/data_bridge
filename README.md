## Data Bridge

  基于Jruby 和 Java Jdbc 库获取自定义数据库接口的Sql数据源，输出到指定接口的中间桥接程式.

####需求环境

  * JRuby 1.7.10
  * Oracle-jdbc6.jar
  * Jdbc-sqlite3-3.7.2.1
  * Sequel-4.10.0
